# Starship Cockpit Extension

This is a standalone Chrome extension. 

## Installation
1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select this `extension` folder

## Note
You do **NOT** need to run `npm run build` or any Vite commands for this extension to work. It uses raw CSS and JS for maximum compatibility.
